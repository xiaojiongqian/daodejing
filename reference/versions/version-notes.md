# 《道德经》版本与出土文献笔记

## 1. 传世系统

### 王弼本

- 王弼，三国魏人，所注《老子道德经注》是后世影响最大的《老子》注本之一。
- 王弼系统重在玄学义理，常以“无”“本”“自然”等概念解释《老子》。
- 后世通行的八十一章结构，很大程度上以这类传世本为基础。

### 河上公本

- 《老子河上公章句》是另一套非常重要的传世注本。
- 阅读重点常落在治国、治身、养生、清静等方向。
- 与王弼注相比，河上公传统更容易把政治秩序、身心修养和道教化解释连在一起看。

## 2. 出土文献

### 马王堆帛书《老子》

- 1973 年长沙马王堆汉墓出土帛书，其中包括《老子》甲、乙两种抄本。
- 甲本通常被认为较早，乙本稍晚；二者都早于多数传世定本。
- 帛书本常见特点之一是“德经”在前、“道经”在后，因此也常引发“道德经 / 德道经”的讨论。
- 帛书文字与传世本存在异文，例如“常 / 恒”“无 / 亡”等处，需要逐章比对，不宜只凭单句判断。

### 郭店楚简《老子》

- 1993 年湖北荆门郭店一号楚墓出土竹简，包含《老子》甲、乙、丙三组材料。
- 郭店简通常被视为目前所见较早的《老子》传抄材料之一。
- 简本并非完整八十一章系统，篇章选择、顺序和文字都与传世本不同。
- 对思想史很重要：它提醒我们，《老子》可能经历了较长时间的编纂、流传和定型过程。

## 3. 阅读时的版本意识

- 不同版本的差异不只是“错字”，也可能反映战国、秦汉之间文本流传的层累。
- 做讲解或改写时，可先用传世通行本保持可读性；做研究或严肃注释时，应列出异文。
- 不建议把某一出土本直接说成“唯一原版”。更稳妥的说法是：它们提供了更早阶段的文本面貌。

## 4. 可优先整理的对照主题

| 主题 | 传世本重点 | 出土本值得注意处 |
| --- | --- | --- |
| 章序 | 道经在前，德经在后 | 马王堆帛书多见德经在前 |
| 用字 | 常、无、天地等通行写法 | 可能见恒、亡等异文 |
| 第十九章 | “绝圣弃智”“绝仁弃义”等 | 郭店简相关文字与传世本差异较受关注 |
| 政治思想 | 无为而治、小国寡民 | 简帛材料可帮助观察早期语境 |
| 修身思想 | 致虚守静、少私寡欲 | 河上公注与帛书背景可并读 |

## 5. 本仓库的文本整理结果

本目录已据上述版本意识，把"资料起步版"整理为多源交叉校订的全文：

- **全文底本**：以王弼《老子道德经注》（华亭张氏本）为底本，参校 ctext.org 通行本与河上公章句本。全经八十一章中，王弼本与 ctext 通行本经文 **49 章完全一致**；其余各章异文（多为虚词、通假、个别脱字）逐章列于 [版本异文校记](variant-apparatus.md)。
- **6 处通行本校改**：正文面向讲解与改写，于第 2、3、16、25、78、80 章采用通行本习见读法（如第 16 章"公乃王"、第 25 章"天地母"、第 78 章"以其无以易之"），王弼底本原读与依据逐条见 [校记第二节](variant-apparatus.md)。
- **电子本须校**：在线电子本偶有脱误（如 ctext 本第 3 章"使心不乱""使天知者"），故选转写更严谨的王弼本为底本；严肃引用仍以纸本校勘本为准。
- **出土文献**：帛书甲乙本、郭店楚简的高置信度对照见 [出土文献选辑](../texts/mawangdui-guodian-selected.md)；其精校全文须据高明《帛书老子校注》、荆门市博物馆《郭店楚墓竹简》等纸本。
