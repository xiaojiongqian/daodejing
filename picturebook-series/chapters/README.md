# 八十一章章节稿

本目录存放《道德经》八十一章逐章绘本材料。每个章节文件都先按统一模板留位，后续逐章补充。

`ch001`～`ch081` 都遵循单页短文本规范：每页一般 2～3 句，中文最好不超过 150 个字，英文最好不超过 750 个字节，并保持适合朗读和 TTS 转换。

英文 `Original Text` 使用 James Legge (1891) 译本作底本，但章节页内不显示译文出处、公共领域说明或 source text 路径，避免 TTS 读出非正文内容。

后续章节的典故和故事选材，应明显提高西方及中国以外材料的比重，目标约占典故/故事内容的 70%。比例服务于内容质量，不机械凑数；所有外部典故仍需可靠、贴合章义，并能改写成儿童可理解的叙事。

如果创作时引入 `reference/allusion/` 尚未收录的新典故、新史事或新公共领域故事，必须同步补入 `reference/allusion/` 对应分类文档。新增典故要先查重，再按该目录体例写明出处、关联章句、典故复述、义理说明与绘本用法；材料必须找得好、找得恰当、有吸引力，不能为了凑比例硬套。

章节总表见 [chapter-index.md](chapter-index.md)。

## 每章基本结构

每章或拆分后的每篇只保留清晰的纯文字稿，不使用表格，不写画面描述。每篇通常为 15～20 页。单篇完整章节或拆分后的第一篇，通常包括：

1. 原文：放本章《道德经》原文。
2. 释义：用孩子能理解的话解释。
3. 正文：用故事、典故或讲述展开。
4. 回扣：回到原文，点明故事和章句的关系。

若某章典故内容较多，可以拆成多篇，如 `ch002-1.md`、`ch002-2.md`。第一篇必须完整呈现本章原文及整体解释，不得把原文拆散；后续篇可以不再重复原文，可使用“典故引入 / 正文 / 回扣”等结构继续延展。

## 使用方式

- 新写一章前，先复制 `_chapter-template.md` 的结构到对应 `ch###.md`；若拆成多篇，则使用 `ch###-1.md`、`ch###-2.md`。
- 找不到故事时，也可以把典故、案例或概念解释写进“正文”部分。
- 每章完成后，在 `summaries/` 建立摘要，并更新 `RECENT_EVENTS.md`。
